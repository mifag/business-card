Программирование

В свободное от работы время занимаюсь кодингом. Обучаюсь разработке web-приложений. Интересуюсь как frontend, так и backend разработкой, но больше интересует все-таки backend. Изучаю Java. Пример кода - CRUD приложение , которое использует четыре основных функции для работы с базой данных: Create - создать, Replace - заменить, Update - обновить, Delete - удалить. Для примера я использовал миди-клавиатуры и их владельцев. Для управления базой данных я использовал систему PostgreSQL. Мой код можно посмотреть здесь: Demo-app.
Frontend здесь является отдельным репозиторием-субмодулем всего приложения.
